JavaScript js 學習 筆記
2022/05/12

1-1. JavaScript是甚麼

JavaScript簡稱JS，接下來為了方便都會以JS敘述，JS是一種與電腦溝通的語言也叫程式語言，因為電腦不是人，沒辦法用我們平常的語言溝通，像我現在打的中文他也不知道是啥意思，所以可以趁現在罵他(誤)，那我們要怎麼跟電腦溝通呢? 就是要使用到程式語言，程式語言有分很多種例如C++、PHP、Python等等，就像人類語言有分英文、中文、韓文等等的概念，而JS就是其中一個程式語言但特別的是他是專門使用在網頁上的。

1-2:JavaScript的用途

網頁有分三大要素，HTML、CSS以及JS，HTML是網頁中不可或缺的要素，HTML就像是人，CSS為外觀穿著或化妝，JS則像是動作表現如跳舞運動，所以人一定要存在否則其他皆不會存在，有了人就可以選擇要不要穿搭化妝或是要不要運動跳舞，那我比較廢我啥都不想做可以嗎? 當然可以啊，但你就是變一個很簡單的人?(很怕用錯詞)，所以網頁也是這樣JS就是讓你的網頁能動起來的功能。

1-3:JavaScript的使用

例如預約1922疫苗有預約成功時，訊息會跳出預約成功的訊息，還有我們瀏覽網頁時突然出現的滿版廣告，電商下單後顯示加入購入車訊息，另外還有當離線時google出現的恐龍遊戲也是使用JS完成的，但JS只能完成這些事嗎?當然不是，以下再舉例幾個:

  1. 在網頁中插入新的HTML，修改現有的網頁內容和網頁的樣式。
  2. 回應用戶的行為，例如鼠標的點擊或移動特效
  3. 動態的更新內容、控制多媒體、動畫
  4. 建立數據庫

以上為初學者JS較為理解的例子，另外還有更多功能使用，待未來好好學習
