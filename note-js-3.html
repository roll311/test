2.JavaScript核心概念

JavaScript js 學習 筆記
2022/05/12

2-1:基本語法範例

let chickenFarm = '小雞'

chickenFarm中文為養雞場，上述程式碼簡單來說就是把小雞丟到養雞場裡面，為甚麼要這樣做呢？因為有養雞場後才能管理小雞。

JS也是這樣，需要先提供一個暫時儲存資料的地方那個地方就是chickenFarm，而chickenFarm裡面放的資料就是小雞，這就是上面程式碼在做的事情

2-2:環境設定

要執行上述的程式碼就需要有一個開發環境，讓你輸入程式碼並且執行外還可偵測錯誤，知名的開發環境有VS Code以及 Code Pen在這裡我會使用Code Pen作範例

先來簡單介紹一下Code Pen ，註冊登入後點選Pen，我們接下來會使用到的是JS的區塊，跟左下角小小的Console（主控台），主控台的作用是讓開發人員能觀察系統的內部訊息。我們會在這裡即時測試 JavaScript 程式碼。

2-3:基本語法拆解

let chickenFarm = '小雞'

接下來拆解剛剛展示的程式碼

宣告類型

  1. 變數let、var
  2. 常數const

為甚麼要宣告呢?因為要告訴電腦說我丟進去的資料，是可以被更改的還是不可以被更改的。

像上述程式碼變數let就是告知電腦我這個資料是可以被更改的，例如養雞場原本只有一隻小雞，之後我又可以丟入好幾隻小雞。

若改為常數const則養雞場只能有當初放的小雞，之後不能再放其他小雞。

宣告大致是這樣，另外需特別注意變數var容易造成程式碼的錯誤，所以接下來都使用let這個宣告變數。 若不確定該使用變數或常數，請直接使用常數除非預期會更動。

變數命名

變數名稱可隨意，例如我想養馬變數就改為horseFarm(養馬場)，但有發現一個特別的地方嗎，英文的第二個單詞首字都是大寫，這是因為採用了駝峰命名法，也就是將第一個單字之外的接下來每一個字首大寫。

你會問如果我變數只有一個單詞呢?也可以，命名長度不限制，但須要遵從一些規則:

  1. 需是英文字
  2. 可加上$、_ 或數字，如$chickenFarm、chicken_Farm，但以小寫英文字母開頭仍是最標準的做法
  3. 開頭不許加上數字，如：9chickenFarm
  4. 大小寫有差異，如chickenfarm跟chickenFarm就是不同的變數
  5. 不可是JavaScript 保留關鍵字，如const、let等等

運算子

上述程式語法內的＝就是運算子內的一種，在養雞場的例子來說他像是貨車，把買來的小雞送到養雞場去。

關於運算子有以下幾種：

  1. 算術運算子
  2. 賦值運算子
  3. 比較運算子
  4. 邏輯運算子

接下來來介紹各種常用的運算子

算數運算子
＋ 加法
— 減法
＊ 乘法
／ 除法
％ 取餘數，如14/3=4.....2
餘數就是2
＋＋ 將值加1，如3++就是3+1=4
–– 將值減1，如3-- 就是3-1=2
＊＊ 計算Ｘ的Ｙ次方，如5＊＊3
就是5＾3＝125
賦值運算子
＝ 就是我們常用的=
有分配以及指定的意思
＋＝ x=x+y的縮寫
-＝ x=x-y的縮寫
/＝ x=x/y的縮寫
%＝ x=x%y的縮寫
比較運算子
＝＝＝ 嚴格的等於
！＝＝ 嚴格的不等於
> 大於
< 小於
>= 大於等於
<= 小於等於
邏輯運算子
&& 與，每個值皆為true時才返回true
|| 或，只要其中有值為 true 即返回 true
! 非，將值轉換布林值後返回相反值
