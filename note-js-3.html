2.JavaScript核心概念

JavaScript js 學習 筆記
2022/05/12

2-1:基本語法

看不懂?沒關係，先看就對了

let chickenFarm = '小雞'

chickenFarm中文為養雞場 用我們聽得懂得語言來說就是把小雞丟到養雞場裡面!為甚麼要這樣做呢，因為有養雞場後才能管理小雞。

JS也是這樣，需要先提供一個暫時儲存資料的地方那個地方就是chickenFarm，而那個地方裡面放的資料就是小雞，這就是上面程式碼在做的事情

2-2:環境設定

要執行上述的程式碼就需要有一個開發環境，讓你輸入程式碼並且執行程式碼此外還可偵測錯誤，知名的開發環境有ＶＳＣＯＤＥ以及ＣＯＤＥ。ＰＥＮ在這裡我會使用code.pen

先來簡單介紹一下code.pen ，註冊登入後點選ＰＥＮ，我們接下來會使用到的是ＪＳ的區塊 跟左下角小小的Console（主控台），主控台的作用是讓開發人員能觀察系統的內部訊息。我們會在這裡即時測試 JavaScript 程式碼。

2-3:基本語法拆解

let chickenFarm = '小雞'

那接下來來一一拆解剛剛展示的程式碼

let->宣告

chickenFarm->變數命名

=->運算子

'小雞'->資料型別

宣告:

宣告有分兩種類型

  • 變數let、var
  • 常數const

為甚麼要宣告呢?因為要告訴電腦說我丟進去的資料，是可以被更改的還是不可以被更改的。

像上述程式碼變數let就是告知電腦我這個資料是可以被更改的，例如養雞場原本只有一隻小雞，之後我又可以丟入好幾隻小雞。

若改為常數const則養雞場只能有當初放的小雞，之後不能再放其他小雞。

宣告大致是這樣，另外有幾點特別注意: 因變數var容易造成程式碼的錯誤，所以接下來都使用let這個宣告變數。 另外若不確定該使用變數或常數，請直接使用常數除非預期會更動。
